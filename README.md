# Contact Card

A small Next.js (TypeScript) site that shares two people's contact details.
Each card shows a first name, last name, and an email address with an envelope
icon; clicking the icon or address opens the visitor's mail client.

## Edit the contacts

All content lives in [`data/contacts.ts`](./data/contacts.ts). Replace the
placeholder names and emails there — nothing else needs to change.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
```

## Deploy to Vercel

Push this folder to a Git repository and import it at
[vercel.com/new](https://vercel.com/new), or run:

```bash
npx vercel        # preview deploy
npx vercel --prod # production deploy
```

Vercel auto-detects Next.js — no extra configuration required.
